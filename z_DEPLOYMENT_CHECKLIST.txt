DOWE "The Mini Giant" - Platinum Standard Deployment
===================================================

□ STEP 1: Copy Files
  □ Copy all .nss files to your module
  □ Copy cleanup_config.2da to override folder

□ STEP 2: Compile
  □ Batch compile in toolset
  □ Verify no errors

□ STEP 3: Assign Module Events
  □ OnModuleLoad = area_on_load
  □ OnModuleHeartbeat = area_dispatcher
  □ OnPlayerUnacquireItem = area_on_drop
  □ OnCreatureDeath = area_on_death

□ STEP 4: Assign Area Events (ALL AREAS)
  □ OnAreaEnter = area_on_enter
  □ OnAreaExit = area_on_exit
  □ OnAreaHeartbeat = area_heartbeat

□ STEP 5: Configuration
  □ Set DOWE_DEBUG = 1 (for initial testing)
  □ Set DOWE_SQL_USE_EXTERNAL = 0 or 1
  □ Tune DOWE_ENC_PLAYERS_PER_TICK if needed

□ STEP 6: Test
  □ Enter area - check manifest count
  □ Drop item - verify self-registration
  □ Exit area - verify zero-waste shutdown
  □ Check debug messages
